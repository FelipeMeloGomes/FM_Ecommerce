# 🛒 Ecommerce Full Stack com Next.js

Projeto de e-commerce desenvolvido para aprimorar minhas habilidades como **desenvolvedor Full Stack**, com foco no ecossistema do **Next.js** e no uso de tecnologias modernas que aumentam a produtividade, segurança e qualidade do código.

O projeto simula um ambiente real de mercado, incluindo autenticação de usuários, integração com pagamentos e gerenciamento de produtos via CMS.

---

## 🌐 Aplicação em Produção

👉 Acesse o projeto online:  
**https://fm-ecommerce-jade.vercel.app/**

---

## 🚀 Tecnologias

- **Next.js**
- **TypeScript**
- **Tailwind CSS**
- **Sanity CMS**
- **Stripe**
- **Clerk** (autenticação e registro de usuários)

---

## ▶️ Getting Started

Para iniciar o servidor de desenvolvimento, execute:

```bash
npm run dev
# ou
yarn run dev
# ou
pnpm run dev
# ou
bun run dev
v
```

## 📦 Adição de Produtos ao Banco de Dados

Para adicionar produtos ao banco de dados utilizando o arquivo de seed presente no projeto (Sanity CMS), execute o comando abaixo:

```bash
pnpm dlx sanity@latest dataset import seed.tar.gz
```
